API

Welcome to Pacifica's API documentation. We offer complete REST and Websocket APIs to suit your trading needs. The documentation includes clear descriptions, parameter requirements, example responses, and code snippets to help developers quickly implement these endpoints in their applications. You can also use the API via the Pacifica Python SDK: https://github.com/pacifica-fi/python-sdk For any feedback, suggestions, and additional help with integration into Pacifica's API, please let us know in our discord API channel!
